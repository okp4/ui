# Stepper

> Allows to complete steps

[![stability-unstable](https://img.shields.io/badge/stability-unstable-yellow.svg)](https://github.com/emersion/stability-badges#unstable)

## Description

The `Stepper` component allows a user to provide `Steps` which are composed of `JSX.Element` and which then allows him to navigate between these steps.
